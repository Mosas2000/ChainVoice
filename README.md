# ChainVoice Smart Contracts

Decentralized social messaging system on Stacks blockchain.

## Contracts

### Profiles Contract
- User profile creation and management
- Follow/unfollow functionality
- Profile data storage

### Messages Contract
- Public message posting
- Direct messaging
- Message retrieval and filtering

## Technology
- Clarity smart contracts
- Stacks blockchain
- Testnet and mainnet deployment

## Setup
npm install
npm run test

## Deployment
npm run deploy:testnet
npm run deploy:mainnet
