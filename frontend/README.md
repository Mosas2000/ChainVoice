# ChainVoice

Decentralized social messaging platform built on Stacks blockchain.

## Features
- On-chain user profiles
- Public message posting
- Direct messaging
- Follow/unfollow users
- Message reactions
- Professional UI with shadcn/ui

## Tech Stack
- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components
- Stacks Connect
- React Router

## Setup
npm install
npm run dev

## Build
npm run build
